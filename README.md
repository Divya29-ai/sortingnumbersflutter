Sorting Algorithm Visualizer 

This project is a Flutter-based Sorting Algorithm Visualizer that demonstrates how sorting algorithms work in real time.
It visually represents array elements as bars and shows step-by-step sorting using animations.
The application supports multiple algorithms like Bubble Sort, Selection Sort, and Merge Sort.
Users can control array size and animation speed for better understanding.
It also displays performance metrics such as comparisons and swaps during execution.
The project is built using Flutter and Dart with an interactive and user-friendly UI.
This project helps in understanding sorting concepts, recursion, and algorithm efficiency.
